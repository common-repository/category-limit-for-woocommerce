== Description ==

Woocommerce toolkit to only allow purchase from one or more parent categories for each order.


= Plugin Features =

* Limit maximum number of categories for each order.
* Custom name for category on error (shop, restaurant, etc).
* Exclude categories from the limit


== Installation ==

1. Upload the plugin.
2. Activate plugin
3. Go to Woocommerce -> Settings -> Category Limit.
